# Privacy Policy

## Overview

HeartFix does not provide an account system or a first-party backend service in this public repository.

## Data Handling

- Relationship text entered by the user is processed on the device.
- Unlock state is stored locally using AsyncStorage.
- No first-party server is included in this repository.

## Third-Party Services

- If you configure a DeepSeek API key, analysis requests are sent directly from the app to DeepSeek.
- If you use the premium unlock flow, wallet connection and transaction signing are handled through the user's Solana wallet and Solana RPC infrastructure.

## Developer Responsibility

Anyone deploying or modifying this app is responsible for providing their own API credentials, wallet recipient address, and jurisdiction-specific privacy disclosures where required.
